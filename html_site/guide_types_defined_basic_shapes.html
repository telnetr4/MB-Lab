Types defined by basic shapes

While the previous types defined by the proportions are designed for ideal people and comics characters, the basic types are statistically very common in the real people. Most of the bodies can be obtained as combination of these sets. They are defined by a mix of skeleton proportions and mass/muscle percentages, developed upon anthropometric statistics and artistic anatomy studies.

The library for these types includes the following items for both female and male characters:

  1. Lightbody
  2. Normobody
  3. Heavybody
  4. Vigorous01
  5. Slender01
  6. Stocky01
  7. Stocky02

Other basic types be labelled using the name of the predominant shape:

  • Hourglasses
  • Rectangles
  • Pears
  • Triangles

Other types, not describable with adjectives, are labelled "common" suffixed with a progressive number.
