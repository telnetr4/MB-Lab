Dwarf male (M_FT02.50)

M_FT02.50 is a template for fantasy male dwarf.

The mesh is derived from M-CA01 so it includes the main muscles and tendons and an advanced topology for the face expressions. The model provides teeth and detailed eyes with separated cornea and eyelashes.
