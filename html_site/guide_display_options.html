Display options

In order to check the appearance of the skin, including displacement and other details, it's required to enable some effects in realtime. These features are very CPU expensive so they are disabled by default.

The display option panel permit to control the view preferences.

The button "Enable displacement preview" will show the displacement directly in the viewport, in realtime. Anyway, since usually the displacement is matter of little details, it's also required to press the button "Enable subdivision preview". This button subdivides the model in realtime, so it has enough resolution to show the displacement effects.

Subdivision in realtime is very CPU consuming and should be used temporarily only for a quick preview.

The button "Disable corrective smooth" disable the corrective smooth modifier, in order to see the result of the rigging with pure weights only.
