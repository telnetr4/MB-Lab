Elf male (M_FT01.50)

M_FT01.50 is a template for fantasy male elf.

The topology is derived from M-CA01 so it includes the details for main muscles and tendons and an advanced structure for the face expressions. The model provides teeth and detailed eyes with separated cornea and eyelashes.
