Mixing types, special types and phenotypes

Types, special types and phenotypes can be mixed in order to create intermediate types.

By default, the Mix option is disabled. This means that each time a new type or phenotype is selected, it replaces the current one.

Enabling the Mix option, the current character will be mixed with the next one.

A typical procedure to use the Mix is illustrated in the following example, mixing the extreme overweight type with the extreme bodybuilder type in order to obtain a "overweight muscled" type:

  1. With the "Mix" disabled (in order to overwrite any previous character) load the overweight using the type selector.
  2. Enable the "Mix" option
  3. Load the body-builder using the type selector.

When the "Mix" option is enabled, loading more time the same type will increase its influence in the final character. For example if we load the body-builder consecutively for two times, the resulting character will be more muscled.
