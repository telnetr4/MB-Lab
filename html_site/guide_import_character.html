Import characters from json data

With the "import character" button you can import the json data from a file (previously exported from a lab session using the "export character" button), in order to transform the current character in the one described in that file.

This function is very useful and permits to

  • Import a character created with the finalization backup.
  • Import a character exported with a previous version of the lab. This will automatically upgrade the character.
  • Convert characters: for example you can save a json from an human character and then import it on a elf character: the system will create the correspondent elfic version of the human character.
