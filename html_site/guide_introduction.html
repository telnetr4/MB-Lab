Introduction

ManuelbastioniLAB is a project aimed to create a scientific and powerful 3D humanoids editor. The result of the research is released as a free and open source advanced addons for Blender:

This software is based on the long experience gained during the realization of various 3d graphics projects, and collects python scripts, new algorithms and original prototypes models developed by Manuel Bastioni.
